# mccg

一个MC指令生成器。

### ⚠️⚠️⚠️本项目不考虑兼容性，请使用最新版本的浏览器⚠️⚠️⚠️

## 如何使用

安装[Node.js](https://nodejs.org)和[npm](https://npmjs.com)(npm应该是Node.js自带的), clone这个仓库, 第一次开服运行：
```bash
npm install
```
开服就运行：
```bash
node ./server/server.mjs
```
